## 书源导入教程

2.0及3.0老版：

>打开阅读app，在书源界面选择从网络导入书源，把下面的链接粘贴上去即可。

3.0新版：

>1) 同老版本导入
>
>2) 阅读app中，选择订阅页面（软件下方左数第三个图标）。点击规则订阅即可添加相关订阅
